Million Songs API

This API is used with the Million Song Dataset made available by the CORGIS dataset project to provide users with information related to songs and their affiliated artists. The purpose of the API is to deliver to users both basic and advanced features based on this dataset through a Web app.
More information: https://helloreverb.com

get /getArtistsByPopularity
(getArtistsByPopularityGet)
Retrieve an ordering of the artists ranked by their popularity (hotttness index) from more to less popular, with the possibility to subset this order, e.g. the top 50 artists.

Query parameters

hottness (required)
Query Parameter — Popolarity ranking to filter songs by
top (optional)
Query Parameter — Size of the top

Return type

Artist

Example data

Content-Type: application/json
{
  "similar" : 2.302136,
  "familiarity" : 0.08008282,
  "hotttnesss" : 0.6027456,
  "terms" : "terms",
  "latitude" : 1.4658129,
  "name" : "name",
  "location" : 5,
  "id" : "id",
  "terms_freq" : 7.0614014,
  "longitude" : 5.637377
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json
  • text/csv

Statistics

Up 
get /getStatistics
(getStatisticsGet)
Get descriptive statistics (mean, median, standard deviation) for the popularity of the songs for a particular artist with an optional filter by year.

Query parameters

name (required)
Query Parameter — The name of the artist
year (optional)
Query Parameter — Year to filter by

Return type

Statistics

Example data

Content-Type: application/json
{
  "median" : 6.0274563,
  "mean" : 0.8008282,
  "standard_deviation" : 1.4658129
}
